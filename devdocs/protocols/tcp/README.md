# OBI TCP based Protocols

This directory contains documentation for the supported protocol parsers that OBI provides based on raw TCP packets.

## Table Of Contents

- [Aerospike](aerospike.md): Aerospike native client protocol parser.
- [Couchbase](couchbase.md): Couchbase (Memcached Binary Protocol) parser.
- [Kafka](kafka.md): Kafka protocol parser.
- [Memcached](memcached.md): Memcached text protocol parser.
- [MQTT](mqtt.md): MQTT protocol parser.
- [NATS](nats.md): NATS protocol parser.
- [AMQP](amqp.md): AMQP 1.0 protocol parser.
- [SunRPC](sunrpc.md): ONC RPC (Sun RPC) protocol parser.
- [PostgreSQL](postgres.md): PostgreSQL protocol parser.
- [New Tracer](new-tcp-tracer.md): how to add a new TCP protocol based BPF tracer to OBI.
